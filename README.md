# FlfCreator

FlfCreator is a simple library of helper functions for creating fixed length record files.

## Installation

Add this line to your application's Gemfile:

    gem 'flf_creator'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install flf_creator
